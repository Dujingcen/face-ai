# 功能 #

1. 人脸检测、识别（图片、视频）
2. 更换背景图（例如给任务换一张海景背景图）
3. 图片合成（多张照片合成）
4. 证件照更换背景色
5. 年龄识别
6. 性别识别
7. 图片修复（可用于水印去除）
8. 图片分类
9. 老照片
10. 素描


**查看功能预览↓↓↓**

# 开发环境 #

- Windows 10（x64）
- Java 1.8
- OpenCV 3.4.3
